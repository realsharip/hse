## 📋 Task Description

Express 1/N as sum of two unit fractions: 1/N = 1/A + 1/B where A < B. Output A+B. For N ≤ 12.
## 📥 Input Format

An integer N (2 ≤ N ≤ 12).
## 📤 Output Format

Sum A+B for the decomposition.
