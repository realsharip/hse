## 📋 Task Description

Tank capacity C liters. Currently has W liters. Adding A liters per minute for M minutes. How much overflows?
## 📥 Input Format

Four integers: C (capacity), W (current), A (rate), M (minutes).
## 📤 Output Format

Amount of overflow (0 if no overflow).
