## 📋 Task Description

Calculate F(n) mod M where F(n) = F(n-1) + F(n-2), F(1)=1, F(2)=1. Only for n ≤ 10.
## 📥 Input Format

Two integers: n (≤10), M (modulus).
## 📤 Output Format

F(n) mod M.
