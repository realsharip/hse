## 📋 Task Description

Calculate total length of Morse code for a two-digit number. Digits: 0=5, 1-5=5-1, 6-9=4-5 dashes/dots.
## 📥 Input Format

A two-digit integer N.
## 📤 Output Format

Total number of symbols (dots and dashes) in Morse representation.
