## 📋 Task Description

A ferris wheel has N cabins. You start in cabin 1. After K rotations of M cabins each, which cabin are you in?
## 📥 Input Format

Three integers: N (total cabins), K (rotations), M (cabins per rotation).
## 📤 Output Format

Final cabin number (1-indexed).
## 💡 Examples

### Example 1

**Input:**
```
10
3
3
```

**Output:**
```
10
### Example 2

**Input:**
```
8
5
3
```

**Output:**
```
8
### Example 3

**Input:**

12

```

```
12
