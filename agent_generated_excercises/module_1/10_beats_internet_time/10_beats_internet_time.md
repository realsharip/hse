## 📋 Task Description

Convert time to Swatch Internet Time (.beats). One day = 1000 beats, starting at midnight UTC+1.
## 📥 Input Format

Two integers: H (hours UTC), M (minutes).
## 📤 Output Format

Internet time in beats (0-999).
