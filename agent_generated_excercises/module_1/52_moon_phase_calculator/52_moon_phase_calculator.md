## 📋 Task Description

Simplified moon phase: cycle is 29 days. Day D since new moon, what phase? 0-7=new, 8-14=first quarter, 15-21=full, 22-29=last quarter.
## 📥 Input Format

## 📤 Output Format

Phase number: 0=new, 1=first quarter, 2=full, 3=last quarter.
