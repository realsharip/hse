# Title: Base64 Padding Count
# Topic Tags: base64, padding

## Task
How many padding '=' characters needed to encode N bytes in base64?

## Input
Number of bytes N.

## Output
Number of padding characters (0, 1, or 2).

## Examples
Input:
3
Output:
0

Input:
4
Output:
2

Input:
5
Output:
1
