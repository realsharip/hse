## 📋 Task Description

Gear A has Ta teeth, Gear B has Tb teeth. If A rotates Ra times, how many degrees does B rotate?
## 📥 Input Format

Three integers: Ta (teeth A), Tb (teeth B), Ra (rotations of A).
## 📤 Output Format

Degrees B rotates (integer).
