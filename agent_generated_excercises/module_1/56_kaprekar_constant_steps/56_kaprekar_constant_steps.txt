# Title: Kaprekar Constant Steps
# Topic Tags: kaprekar, constant

## Task
For 3-digit number, apply Kaprekar process: sort digits desc - sort asc. Count steps to reach 495.

## Input
A three-digit number with at least two different digits.

## Output
Steps to reach 495.

## Examples
Input:
100
Output:
5

Input:
234
Output:
3

Input:
495
Output:
0

## Constraints
- Use only Module_1 concepts: arithmetic operations, modulo, integer division, digit extraction.
- No loops, no conditionals, no lists/tuples/dicts/sets, no user-defined functions.
- Prefer integer math (`//`, `%`) and arithmetic identities over any form of branching.
