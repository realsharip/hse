## 📋 Task Description

Taxi charges: B rubles base fare, then P kopecks per 100 meters. Calculate fare for D meters.
## 📥 Input Format

Three integers: B (base fare in rubles), P (kopecks per 100m), D (distance in meters).
## 📤 Output Format

Total fare in rubles and kopecks.
