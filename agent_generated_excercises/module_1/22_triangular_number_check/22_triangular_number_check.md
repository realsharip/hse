## 📋 Task Description

Check if N is a triangular number (sum of first k integers for some k). Output k or 0.
## 📥 Input Format

An integer N (1 ≤ N ≤ 100).
## 📤 Output Format

k if N is triangular (N = k(k+1)/2), otherwise 0.
## 💡 Examples

### Example 1

**Input:**
```
10
```

4

```

**Output:**
```
