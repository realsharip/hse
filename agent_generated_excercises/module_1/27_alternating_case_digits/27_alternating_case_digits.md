## 📋 Task Description

For a three-digit number, output digits separated by alternating 'A' and 'B'.
## 📥 Input Format

A three-digit integer N.
## 📤 Output Format

Formatted string with pattern: dAdBd where d are the digits.
## 💡 Examples

### Example 1

**Input:**

**Output:**

```
456
```

**Output:**
```
4A5B6
### Example 3

789
