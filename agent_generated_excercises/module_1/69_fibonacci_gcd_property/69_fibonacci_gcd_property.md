## 📋 Task Description

For Fibonacci numbers, GCD(F(m), F(n)) = F(GCD(m,n)). Verify for given m, n ≤ 10.
## 📥 Input Format

Two integers m, n (≤ 10).
## 📤 Output Format

F(GCD(m,n)).
