## 📋 Task Description

Center a number in a field of width W using asterisks. If even padding needed, put extra on right.
## 📥 Input Format

Two integers: N (number to display), W (field width ≥ number of digits in N).
## 📤 Output Format

The centered string with asterisk padding.
## 💡 Examples

### Example 1

**Input:**
```
42
6
```

**42**

```

**Output:**

**7**
